# Nestkin Website

This folder contains a lightweight static website for Nestkin.

Files:
- `index.html` for the home page
- `privacy.html` for the privacy policy
- `terms.html` for the terms of service
- `styles.css` for shared styling

Before publishing:
- Replace placeholder contact details
- Update company or owner information
- Host the folder on a public domain such as Netlify, Vercel, GitHub Pages, or Firebase Hosting
